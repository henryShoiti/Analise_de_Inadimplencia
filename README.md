# Analise_de_Inadimplencia
Análise de risco de crédito end-to-end utilizando o dataset Give Me Some Credit (150k observações). Cobre limpeza de dados, tratamento de valores ausentes, detecção de outliers e análise exploratória de padrões de inadimplência em variáveis financeiras e demográficas. Os resultados são apresentados em um dashboard interativo com Streamlit.
